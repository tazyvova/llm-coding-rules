# llm-coding-rules

Shared coding rules and guidelines for AI coding assistants.

## Files

- `CODE_STYLE.md` — Universal coding standards
- `AGENT.md` — AI-specific behavior rules
- `CLAUDE.md` — Claude Code entry point
- `GEMINI.md` — Gemini entry point
- `CODEX.md` — OpenAI Codex entry point

## Usage

### Git Submodule

```bash
git submodule add https://github.com/YOUR_USERNAME/llm-coding-rules .rules
```

Then symlink or reference files as needed.

### Direct Copy

Copy the files you need to your project root.
